# MITAOE Website

A modern, responsive MITAOE website built with Next.js and Tailwind CSS.

## Introduction

This repository contains the MITAOE website client with dynamic pages, admin APIs, and an easy-to-customize content structure.

## Quick Start

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm start
```

## Notes
- Certain API routes need `MONGODB_URI` in your environment.
- The build has been made resilient when that variable is not available during static generation.
